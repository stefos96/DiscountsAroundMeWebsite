# Discounts Around Me Website
Website featuring Discounts Around Me app for Android Devices

## Deployment

```
git clone
```
```
npm install
```
```
npm install jade-bootstrap
```
```
npm start
```

### Prerequisites
* [Node.js](https://nodejs.org/en/)

## Built With

* [Node.js](https://nodejs.org/en/) - Framework
